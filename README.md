# Godspeed

This repository is a part of a submission to Flipkart's Gridlock Hackathon: "On the right side: Encouraging BMTC drivers to be mindful of road traffic"

## Demo Video

https://vimeo.com/222448655

## Running Instructions

Requirements: `node`, `npm`

To install dependencies, run `npm install`

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
